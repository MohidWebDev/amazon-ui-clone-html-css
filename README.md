# amazon-ui-clone-html-css
Complex Header Engineering: Manages a highly dense navigation bar using Flexbox to align the logo, localized delivery info, a tripartite search engine (select, input, and icon), and account settings.

Grid-Based Product Display: Implements a wrap-around layout using justify-content: space-evenly to create a uniform "shopping card" experience that adapts to the viewport.

Advanced Box Model Usage: Showcases precision in spacing with specific border-hover effects (1.5px solid white) that mimic Amazon's unique interactive feedback system.

Multi-Panel Footer Architecture: Orchestrates four distinct footer sections—ranging from a simple "Back to Top" button to a massive 4-column resource directory—using semantic HTML and background-color nesting.

Iconography & Asset Management: Successfully integrates Font-Awesome for location and shopping cart indicators, while utilizing background-image properties for high-performance logo rendering.
